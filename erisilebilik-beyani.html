Erişilebilirlik Beyanı

Herkes için erişilebilir bir web deneyimi

Son Güncelleme: 15.01.2026

Taahhüdümüz: Web sitemizi engelli bireyler dahil olmak üzere herkes için erişilebilir kılmayı taahhüt ediyoruz. WCAG 2.2 Seviye AA standartlarına uygunluğu hedefliyoruz.

1. Uyumluluk Durumu

Web sitemiz WCAG 2.2 Seviye AA standartlarına büyük ölçüde uygundur.

2. Erişilebilirlik Özellikleri

  • Ekran Okuyucu Uyumluluğu: JAWS, NVDA, VoiceOver uyumlu
  • Klavye Navigasyonu: Tüm öğelere klavye ile erişim
  • Alt Metin: Tüm görsellerde açıklayıcı alt metin
  • Renk Kontrastı: Minimum 4.5:1 kontrast oranı
  • Skip Link: Ana içeriğe hızlı atlama
  • Semantik HTML: Doğru başlık hiyerarşisi
  • Form Etiketleri: Açık ve erişilebilir form alanları

3. Klavye Kısayolları

  • Tab - Sonraki öğeye git
  • Shift+Tab - Önceki öğeye git
  • Enter - Bağlantıyı etkinleştir
  • Esc - Pencereyi kapat

4. Test Edilen Teknolojiler

Tarayıcılar: Chrome, Firefox, Safari, Edge

Yardımcı Teknolojiler: NVDA, JAWS, VoiceOver, TalkBack

5. Bilinen Sınırlamalar

  • Bazı PDF dokümanları tam erişilebilir olmayabilir
  • Harici bağlantılardaki içerikler üzerinde kontrolümüz bulunmamaktadır
